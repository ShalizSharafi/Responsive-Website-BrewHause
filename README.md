## 📌 Overview

**Brewhaus** is a front-end project designed to showcase a coffee shop’s brand, menu, locations, and customer experience.

The design emphasizes:
- Warm, inviting colors
- Smooth hover and scroll animations
- Clean and modern layout
- Fully responsive structure

---

## 🚀 Features

- 📱 **Responsive Design**
  - Works across mobile, tablet, and desktop
- 🎨 **Custom Design System**
  - Tailwind-based theme with custom colors
- ✨ **Animations**
  - Scroll-triggered animations (`fadeUp`)
  - Infinite horizontal carousel
  - Hover transitions and micro-interactions
- 🧭 **Navigation**
  - Desktop navbar + mobile menu layout
- ☕ **Sections Included**
  - Hero section
  - Categories (Coffee, Drinks, Bakery)
  - Popular items
  - Testimonials
  - Locations
  - News/Blog
  - Contact & Footer
- 👤 **Personal Developer Card**
  - Interactive profile section with animations

---

## 🛠️ Tech Stack

- **HTML5**
- **Tailwind CSS (CLI)**
- **Custom CSS Layers**
  - `@layer base`
  - `@layer components`
  - `@layer utilities`
- **CSS Animations**
  - Keyframes (`fadeUp`, `carousel`)
- **Responsive Design**
  - Tailwind breakpoints
